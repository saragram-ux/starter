About

Not a framework. Not too empty either.

Simple by default

Starter gives you a clean base without filling your project with noise.

Easy to shape

Use it as-is, trim it down, or build on top of it over time.

Made for real work

Good naming, clear structure, and a setup you can actually reuse.

What’s included

The basics, done properly.

Layout

  • page-padding
  • container-small / medium / large
  • section spacing
  • simple grid helpers

Typography

  • heading-h1 to heading-h5
  • text-small / base / medium
  • text-muted
  • clean defaults with Inter

Components

  • buttons
  • cards
  • form fields
  • basic utility classes
